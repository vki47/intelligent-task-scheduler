Run It Locally
Activate venv:

venv\Scripts\activate   # Windows
# OR
source venv/bin/activate   # Mac/Linux
Then:

pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
Open:

http://127.0.0.1:8000

To go back:

git checkout main
To return to PR branch:

git checkout codex/update-dsa-enhancements

git fetch origin
git checkout codex/update-dsa-enhancements
